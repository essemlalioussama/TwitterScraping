# TwitterScraping

- Installation : 

        - pip install --no-cache-dir --upgrade -r requirements.txt

        - Docker : 

             Python API : 

                  - docker build --tag python-docker .  

                  - docker run -p 8080:5000 python-docker

              NodeJs API: 

                  - docker build --tag nodejs-docker . 

                  - docker run -p 9000:9000 nodejs-docker
